# TypeScript 入门 - 03：Type和Interface 使用场景


## 使用类型别名的场景

- 定义基本类型的别名时，使用type
- 定义元组类型时，使用type
- 定义函数类型时，使用type
- 定义联合类型时，使用type
- 定义映射类型时，使用type

## 使用接口的场景

- 需要利用接口自动合并特性的时候，使用interface
- 定义对象类型且无需使用type 的时候，使用interface
